## 로고 디자인

데브 고래밥으로 도대체 어떤 느낌의 로고를 만들어야하나? GPT도 포기한 로고 디자인에 몇시간을 날렸는지 모르겠다. AI로고도 맘에 드는게 하나도 없어서 일러스트레이터, 피그마는 다룰줄 모르기에 간만에 픽셀아트로 끄적거려봤다.

처음엔 고래가 당연히 들어가야함에 고래를 그려놨더니 특색도 없고 뭔가 도커생각이 나서 포기했다.
